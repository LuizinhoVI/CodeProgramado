zero dengue

O brasil está cada vez mais aparecendo foco da dengue por conta dos lugares que tendem a conter um pouco de água ao redor, assim já é o suficiente para poder o mosquito colocar os ovos nos locais. Com esse aumento da dengue vai ter muita demanda de funcionários públicos para fazer as visitas dentro de casa, para facilitar o trabalho e acabar com a dengue o Zer@dengue

O projeto se constitui em um simples cadastro que o usuário irá realizar, após feito o cadastro vai para uma página de denúncia assim realizando todo o formulário, entretanto uma equipe de monitoramento vai estar monitorando toda a denúncia do usuário do outro lado do computador.

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML. O Laravel é um framework extremamente fácil de usar, desde a instalação até sua utilização. Tudo pensando para que escreva quantidade mínima de código, ou, se preferir, escreva apenas códigos relacionados a regras do negócio. Atualmente, o Laravel encontra-se em sua versão 5.8, sendo tão completo que passou a ser considerado um verdadeiro ecossistema para o desenvolvimento de aplicações e web services em PHP. Inclusive, o PHP é uma das linguagens de programação mais utilizadas no mundo, sendo utilizada por 79% dos websites.
